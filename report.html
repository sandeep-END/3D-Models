Hierarchy Trees

Controls

  • Escape: Quit
  • 1: set current object as humonoid rightleg
  • 2: set current object as humonoid leftleg
  • 3: set current object as humonoid rightknee
  • 4: set current object as humonoid leftknee
  • 5: set current object as humonoid spinal joint
  • 6: set current object as humonoid righthand
  • 7: set current object as humonoid lefthand
  • 8: set current object as humonoid rightelbow
  • 9: set current object as humonoid leftelbow
  • 0: set current object as humonoid head
  • C: set current object as total humonoid
  • V: set current object as total non humonoid
  • B: set current object as non-humanoid head
  • N: set current object as non-humanoid righthand
  • M: set current object as non-humanoid lefthand
  • Left,Right,Up,Down,Page up,Page down: Rotation Keys for current object
  • F,H,T,G,R,Y: Translation Keys for total models (works only after pressing C or V)
  • A,D,W,S,Q,E: Movement of camera
  • J,L,I,K,U,O: Rotation of camera
  • Z: Toggle Light1
  • X: Toggle Light2
  • P: Toggle Spotlight
